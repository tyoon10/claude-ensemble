Reasoning: The count of points in exactly one list is `|A| + |B| - 2*|A∩B|`. Totals use `e - s + 1` per interval, and the intersection width of an overlapping pair is `min(eA,eB) - max(sA,sB)`.

```python
def symmetric_difference_length(A, B):
    def total(iv):
        return sum(e - s + 1 for s, e in iv)
    i = j = 0
    both = 0
    while i < len(A) and j < len(B):
        s = max(A[i][0], B[j][0]); e = min(A[i][1], B[j][1])
        if s <= e:
            both += e - s
        if A[i][1] < B[j][1]:
            i += 1
        else:
            j += 1
    return total(A) + total(B) - 2 * both
```
