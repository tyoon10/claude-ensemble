Reasoning: A length `b` in `1..len(s)-1` is a border iff `s[:b] == s[len(s)-b:]`. A border is periodic-inducing when its period divides the string length, so I keep the ones where the length is not divisible, and return them sorted ascending. For `len(s) <= 1` there are no borders so I return `[]`.

Spot-checks look right: `"abab"` -> [], `"aabaaab"` -> [3], empty/single -> [].

```python
def borders_non_periodic(s):
    n = len(s)
    out = []
    for b in range(1, n):
        if s[:b] == s[n - b:]:
            if n % b != 0:
                out.append(b)
    return sorted(out)
```
