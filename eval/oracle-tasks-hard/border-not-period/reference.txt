Reasoning: A length `b` in `1..len(s)-1` is a border iff `s[:b] == s[len(s)-b:]`. It is periodic-inducing iff the induced period `p = len(s) - b` divides `len(s)`. I want borders that are NOT periodic-inducing, i.e. `len(s) % (len(s) - b) != 0`. I iterate all proper nonempty prefix-lengths, keep those that are borders and whose period does not divide the length, and return them ascending. Since `b < len(s)`, `len(s) - b >= 1`, so no division by zero. For `len(s) <= 1` the range is empty and I return `[]`.

Spot-checks: `"aaaa"` -> [1] (period 3 doesn't divide 4; periods 2 and 1 do). `"abab"` -> [] (border 2, period 2 divides 4). `"aabaaab"` -> [3] (period 4, 7%4!=0). `"abcabcabc"` -> [3] (b=3 period 6 kept; b=6 period 3 divides 9, dropped). `"aaaaaa"` -> [1,2]. `"ababa"` -> [1,3]. Empty/single -> [].

```python
def borders_non_periodic(s):
    n = len(s)
    out = []
    for b in range(1, n):
        if s[:b] == s[n - b:]:
            if n % (n - b) != 0:   # period is n-b, NOT b
                out.append(b)
    return sorted(out)
```
