Using the closed forms directly.

Part A, D(n,k) = n(1 - (1-1/n)^k):
- D(10,10) = 10(1 - 0.9^10) = 10(1 - 0.3486784401) = 6.513215599
- D(10,1) = 10(1 - 0.9) = 1.0
- D(100,0) = 100(1 - 1) = 0.0  (zero draws => zero distinct)
- D(7,7) = 7(1 - (6/7)^7) = 7(1 - 279936/823543) = 4.620583

Part B, T(n) = n*H_n, using H_n = sum of 1/i for i in 1..n-1 plus the tail:
- T(1) = 1 * 0 = 0.0
- T(2) = 2 * (1 + 1/2) = 3.0
- T(10) = 10 * H_10 = 10 * 2.928968254 = 29.289682540
