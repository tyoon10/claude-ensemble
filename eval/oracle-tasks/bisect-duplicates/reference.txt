Classic binary-search bisection. `insert_left` finds the leftmost slot (mirror of bisect_left): shrink `hi` toward the first index whose value is `>= x`. `insert_right` finds the rightmost slot (bisect_right): advance `lo` past every value `<= x`. Both start with `hi = len(a)` so that "x larger than all" and the empty list correctly yield `len(a)`.

```python
def insert_left(a, x):
    lo, hi = 0, len(a)
    while lo < hi:
        mid = (lo + hi) // 2
        if a[mid] < x:
            lo = mid + 1
        else:
            hi = mid
    return lo

def insert_right(a, x):
    lo, hi = 0, len(a)
    while lo < hi:
        mid = (lo + hi) // 2
        if x < a[mid]:
            hi = mid
        else:
            lo = mid + 1
    return lo
```
